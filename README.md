## Features
- Timer for questions
- Time deduction for wrong answers 
- Progress to next question on right or wrong answers
- Ability to locally store high scores with your initials

## How to Contribute
1. Fork the project to your own machine
2. Refactor code to be further optimized 
3. Commit changes to your own branch
4. Push your work back up to your branch
5. Submit a Pull request so that I can review your changes
